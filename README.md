# homebrew-tap

Homebrew tap for intern tooling.

## Usage

Tap the repo:

```bash
brew tap sbekti/tap
```

Install `internctl`:

```bash
brew install internctl
```

Or install directly from the fully qualified formula name:

```bash
brew install sbekti/tap/internctl
```

## Contents

- `internctl`
  - command-line client for the internal management platform

## Release flow

Formula updates are published automatically from tagged `internctl` releases via GitHub Actions and GoReleaser.
